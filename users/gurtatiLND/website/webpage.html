Memory Game

A simple flip-card memory game using React. The game is designed to train memory by having players flip cards to find matching pairs. It features a clean UI, state management for game logic, and uses React components to handle the card flipping and matching mechanics.

Quiz Game

This project is a quiz game built with React, designed to be an engaging and entertaining way to test and expand your knowledge on a variety of topics. The game presents users with a series of questions sourced from a trivia API and provides feedback on their answers.

My Money

This project is a React application designed to help you save and manage all your expenses in one place. With this app, you can easily add and remove expenses, providing a streamlined way to keep track of your financial activities. Technologies Used: React.JS, CSS, Firebase Authentication, FireStore Database.

SkyStitchers

This project is an educational website designed for children aged 7-12 to learn sewing and crafts in a fun and interactive way. Additionally, the website helps kids find the right workshops in their area, discover craft tutorials, and locate craft events. This website is built using vanilla JavaScript, CSS, and HTML.
